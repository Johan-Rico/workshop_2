## Descripción

Este workshop cubre dos temas fundamentales de Machine Learning:

- **clasificacion/clasificacion.ipynb**: Introducción a modelos de clasificación utilizando el dataset Iris con K-Nearest Neighbors.
- **regresion/regresion.ipynb**: Introducción a modelos de regresión utilizando el dataset California Housing con Regresión Lineal.
